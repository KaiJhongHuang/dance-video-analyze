# Release Notes

## v1.0 (2026-04-03)

首次正式版本發布。

### 核心功能
- YouTube 影片嵌入播放
- 設定開始/結束秒數與八拍數量，自動產生拍子時間軸
- 拍子時間軸以滑桿呈現，每拍有刻度標記，可拖曳跳轉
- 循環播放指定段落
- 播放速度調整（0.25x ~ 1.5x）
- 全螢幕模式（大字拍數顯示 + 浮動控制列）
- 匯出/匯入設定（人可讀的純文字格式）
- 區塊拖曳排序（桌面拖曳 + 手機觸控）
- 鍵盤快捷鍵（Space、方向鍵、L、R、M、+/-、Esc）
- 頁面載入時自動產生預設拍子

### Beta 功能
- 鏡像翻轉（CSS 水平翻轉影片）
- 節拍器音效（Web Audio API，第 1 拍高音、其餘低音）
- 分享連結（設定編碼至 URL 參數，一鍵分享）
- 八拍預覽（新分頁同時顯示 8 拍畫面，4x2 / 2x4 排列）

### 手機支援
- iPhone / 行動裝置響應式設計
- 觸控拖曳排序區塊
- 滑桿觸控操作
- 按鈕與欄位自動適配小螢幕

### 技術
- 純前端靜態網頁，無需後端
- GitHub Pages 部署
- YouTube IFrame API
- Web Audio API（節拍器）
- HTML5 Drag and Drop + Touch Events
