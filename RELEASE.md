# Release Notes

## v1.2 (2026-04-04)

### 新功能
- **循環範圍獨立控制** — 循環起迄時間與數拍範圍分離，可獨立設定不互相干擾
- **進度條綁定循環範圍** — 播放進度條改為對應循環時間而非全曲長度，更直覺
- **數拍浮水印小數顯示** — 浮水印從整數改為 `4.5`、`8.3` 等帶一位小數的顯示，小數字體為整數的 1/3
- **八拍滑桿完整循環** — 滑桿標籤改為 `8,1,2,3,4,5,6,7,8`，8→1 區間也能顯示進度與拖曳
- **數拍浮水印放大** — 浮水印字體加倍（一般 6rem / 全螢幕 8rem）
- **全螢幕浮水印開關** — 全螢幕控制列新增數拍 ON/OFF 按鈕，取代節拍器
- **許願 & 回報 Bug** — Footer 新增 Google Forms 連結

### 改善
- **進度條八拍標記** — 超出循環範圍的八拍組標記自動隱藏，不再溢出
- **八拍滑桿對拍修正** — 修復滑桿顯示與實際拍數差一拍的問題
- **iPhone 直立排版修正** — 修復擷取按鈕（▶| / |◀）在手機直立時溢出邊框的問題
- **iOS 全螢幕安全區域** — 全螢幕控制列加入 `safe-area-inset-bottom` 避免被瀏覽器 UI 遮擋

### 實驗功能
- **自動偵測節拍（本機影片）** — 使用 Web Audio API 頻譜分析自動偵測 BPM 與拍點

## v1.1 (2026-04-03)

### 新功能
- **本機影片支援** — 可從 iPhone / 手機選擇本機影片檔案播放，所有功能（拍子、循環、速度、鏡像、全螢幕）皆可使用
- **BPM 自動計算** — 根據數拍範圍與八拍組數自動算出 BPM，顯示於速度滑桿上方
- **速度滑桿** — 取代原本的速度按鈕，0.5x ~ 1.5x 可拖曳，上方為 BPM、下方為倍率，刻度以 BPM 每 20 為間隔，接近刻度自動吸附
- **數拍範圍欄位** — 開始/結束秒數合併為 `m:ss.s ~ m:ss.s` 格式，附擷取按鈕可直接抓取播放位置
- **自動產生拍子** — 修改數拍範圍或八拍組數時自動觸發，不再需要手動按按鈕
- **八拍預覽下載** — 本機影片的八拍預覽新增下載按鈕，合成為單張圖片

### 改善
- **進度條標籤分離** — 播放進度條的八拍組標籤移至上方，八拍滑桿的拍數標籤維持下方，避免重疊
- **八拍預覽可滾動** — 解決 iOS 瀏覽器上下邊條遮擋問題
- **iPhone 直向預覽修復** — YouTube 八拍預覽在手機直向時也能正確顯示
- **介面簡化** — 移除「產生拍子」「取得目前秒數」「拍數單位」等按鈕/欄位，減少操作步驟
- **鍵盤快捷鍵** — +/- 步進改為 0.05，速度範圍調整為 0.5x ~ 1.5x

## v1.0 (2026-04-03)

首次正式版本發布。

### 核心功能
- YouTube 影片嵌入播放
- 設定開始/結束秒數與八拍數量，自動產生拍子時間軸
- 拍子時間軸以滑桿呈現，每拍有刻度標記，可拖曳跳轉
- 循環播放指定段落
- 播放速度調整（0.25x ~ 1.5x）
- 全螢幕模式（大字拍數顯示 + 浮動控制列）
- 匯出/匯入設定（人可讀的純文字格式）
- 區塊拖曳排序（桌面拖曳 + 手機觸控）
- 鍵盤快捷鍵（Space、方向鍵、L、R、M、+/-、Esc）
- 頁面載入時自動產生預設拍子

### Beta 功能
- 鏡像翻轉（CSS 水平翻轉影片）
- 節拍器音效（Web Audio API，第 1 拍高音、其餘低音）
- 分享連結（設定編碼至 URL 參數，一鍵分享）
- 八拍預覽（新分頁同時顯示 8 拍畫面，4x2 / 2x4 排列）

### 手機支援
- iPhone / 行動裝置響應式設計
- 觸控拖曳排序區塊
- 滑桿觸控操作
- 按鈕與欄位自動適配小螢幕

### 技術
- 純前端靜態網頁，無需後端
- GitHub Pages 部署
- YouTube IFrame API
- Web Audio API（節拍器）
- HTML5 Drag and Drop + Touch Events
