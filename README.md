# 舞蹈練習播放器

> **線上使用：** https://kaijhonghuang.github.io/dance-video-analyze/

一個基於 YouTube 的舞蹈練習工具，幫助舞者精準跳轉到音樂中的特定拍子。

## 功能

### 核心功能
- **YouTube 影片播放** — 貼上任意 YouTube 連結即可載入
- **拍子時間軸** — 設定開始/結束秒數與八拍數量，自動計算每一拍的時間點，以滑桿呈現
- **循環播放** — 在指定區間內反覆播放，適合重複練習特定段落
- **速度調整** — 支援 0.25x ~ 1.5x 播放速度，慢放練舞
- **區塊拖曳** — 各區塊可透過拖曳圖示排序（支援桌面與手機觸控）
- **匯出/匯入設定** — 將當前設定匯出為人可讀的純文字，下次貼回即可還原
- **全螢幕模式** — 影片全螢幕搭配浮動控制列與大字拍數顯示

### Beta 功能
- **鏡像翻轉** — 水平翻轉影片畫面，適合面對螢幕練舞
- **節拍器音效** — 播放時疊加節拍聲，第 1 拍較響，幫助抓拍子
- **分享連結** — 將設定編碼成 URL，傳給舞伴一鍵開啟相同設定
- **八拍預覽** — 開啟新分頁，同時顯示一組八拍中每一拍的影片畫面

## 使用方式

1. 開啟網頁，預設已載入示範影片
2. 播放影片，找到舞蹈段落的起始位置，按「取得目前秒數」填入開始秒數
3. 播放到段落結束位置，再按一次填入結束秒數
4. 輸入這段包含幾個八拍，按「產生拍子」
5. 開啟循環播放，調整速度，開始練舞

### 開始/結束秒數怎麼按？

- **開始秒數** = 第 1 個八拍的第 1 拍的瞬間
- **結束秒數** = 最後一個八拍結束後，假想的「下一拍」的瞬間

## 鍵盤快捷鍵

| 按鍵 | 功能 |
|------|------|
| `Space` | 播放/暫停 |
| `←` / `→` | 上/下一拍 |
| `Shift+←` / `Shift+→` | 上/下一個八拍 |
| `L` | 開關循環播放 |
| `R` | 回到開始 |
| `M` | 鏡像翻轉 |
| `-` / `+` | 減速/加速 |
| `Esc` | 離開全螢幕 |

2. Source 選擇 `main` 分支，資料夾選 `/`
