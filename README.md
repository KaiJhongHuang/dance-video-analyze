# 舞蹈練習播放器

> **線上使用：** https://kaijhonghuang.github.io/dance-video-analyze/

一個基於 YouTube / 本機影片的舞蹈練習工具，幫助舞者精準跳轉到音樂中的特定拍子。

## 功能

### 核心功能
- **YouTube 影片播放** — 貼上任意 YouTube 連結即可載入
- **本機影片播放** — 支援從 iPhone / 手機選擇本機影片檔案播放
- **數拍範圍** — 以 `m:ss.s ~ m:ss.s` 格式設定起迄時間，搭配擷取按鈕可直接抓取播放位置
- **BPM 顯示** — 根據數拍範圍與八拍組數自動計算 BPM
- **八拍滑桿** — 自動計算每一拍的時間點，以單一八拍循環滑桿呈現
- **播放進度條** — 全曲進度條搭配八拍組邊界標記
- **速度滑桿** — 0.5x ~ 1.5x 可拖曳調整，上方顯示 BPM、下方顯示倍率，刻度以 BPM 每 20 為間隔
- **循環播放** — 在指定區間內反覆播放，適合重複練習特定段落
- **全螢幕模式** — 影片全螢幕搭配浮動控制列與大字拍數顯示
- **區塊拖曳** — 各區塊可透過拖曳圖示排序（支援桌面與手機觸控）
- **匯出/匯入設定** — 將當前設定匯出為人可讀的純文字，下次貼回即可還原

### Beta 功能
- **鏡像翻轉** — 水平翻轉影片畫面，適合面對螢幕練舞
- **節拍器音效** — 播放時疊加節拍聲，第 1 拍較響，幫助抓拍子
- **分享連結** — 將設定編碼成 URL，傳給舞伴一鍵開啟相同設定
- **八拍預覽** — 開啟新分頁同時顯示一組八拍中每一拍的畫面（本機影片支援下載合成照片）

## 使用方式

1. 開啟網頁，預設已載入示範影片
2. 播放影片，找到舞蹈段落的起始位置，按 `▶|` 擷取開始時間
3. 播放到段落結束位置，按 `|◀` 擷取結束時間
4. 輸入這段包含幾個八拍組數，拍子會自動產生
5. 開啟循環播放，拖曳速度滑桿調整 BPM，開始練舞

### 數拍範圍怎麼設？

- **開始** = 第 1 個八拍的第 1 拍的瞬間
- **結束** = 最後一個八拍結束後，假想的「下一拍」的瞬間

## 鍵盤快捷鍵

| 按鍵 | 功能 |
|------|------|
| `Space` | 播放/暫停 |
| `←` / `→` | 上/下一拍 |
| `Shift+←` / `Shift+→` | 上/下一個八拍 |
| `L` | 開關循環播放 |
| `R` | 回到開始 |
| `M` | 鏡像翻轉 |
| `-` / `+` | 減速/加速（0.05 步進） |
| `Esc` | 離開全螢幕 |

## 部署

此專案為純靜態網頁，可透過 GitHub Pages 直接部署：

1. 到 repo 的 **Settings → Pages**
2. Source 選擇 `main` 分支，資料夾選 `/`
3. 儲存後等待部署完成
