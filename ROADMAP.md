# Roadmap

## 重要

### A-B 精準微調
在播放進度條上顯示可拖曳的起終點標記，直接拖曳調整數拍範圍，不用手動輸入數字。

### 歌曲庫 / 設定分享社群
使用者可公開分享設定（歌曲 + 拍點 + 速度），其他人一鍵載入。可考慮用 GitHub Gist 或 JSON 檔案作為簡易後端。

### 音樂結構自動偵測
用 Web Audio API 分析音頻波形，自動找出節拍位置和段落邊界，輔助使用者設定數拍範圍。

---

## 次要

### AI 姿態比對
用 MediaPipe / TensorFlow.js 偵測骨架姿勢，和教學影片的動作自動比對打分，給出動作相似度回饋。

### Webcam 並排對比
開啟手機/電腦鏡頭，和影片並排或半透明疊加，練舞時即時比對自己的動作。

### 雙影片同步播放
同時播放兩部影片（老師 vs 學生），比較同一拍的動作差異。

**支援組合：**
| 比較模式 | 同步精度 | 備註 |
|---------|---------|------|
| 本地 vs 本地 | 最佳 | 可截圖 |
| YouTube vs YouTube | 好 | 兩個 iframe |
| YouTube vs 本地 | 可以 | 混搭，持續校正 |

**預期功能：**
- 各自設定開始秒數，對齊同一拍
- 同時播放/暫停/跳拍
- 各自調速
- requestAnimationFrame 持續校正同步偏移

## Nice to Have

### 播放速度漸進模式
自動從慢速開始，每循環 N 次加速 0.1x，逐步提升到原速，適合漸進式練習。
