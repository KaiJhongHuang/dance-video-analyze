# Roadmap

## 規劃中的功能

### 雙影片比較模式
同時播放兩部影片，比較學生與老師在同一拍的動作差異。

**使用情境：**
- 老師示範影片（YouTube 或本地）vs 學生練習影片（手機錄影）
- 左右並排播放，同步控制

**支援組合：**
| 比較模式 | 同步精度 | 備註 |
|---------|---------|------|
| 本地 vs 本地 | 最佳 | 可截圖 |
| YouTube vs YouTube | 好 | 兩個 iframe |
| YouTube vs 本地 | 可以 | 混搭，持續校正 |

**預期功能：**
- 各自設定開始秒數，對齊同一拍
- 同時播放/暫停/跳拍
- 各自調速
- requestAnimationFrame 持續校正同步偏移

**技術挑戰：**
- 精確同步（YouTube IFrame API 時間精度較差）
- 手機同時播兩個影片的效能
- 混搭時的同步校正策略
